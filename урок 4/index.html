html(язык разметки гипертекста)

HTML(от англ. HyperText Markup Language)- этот языкразметки используется для создания веб-страниц и веб-приложений

История

HTML был разрабтан Тимом Бендером-Ли в конце 1980-х годов. Первая версия языка появилась в 1993 году.

основные этапы развития:

  • HTML 1.0 - первый стандартный (заменён болле поздими версями)
  • HTML 4.01- активно использовался до повления HTML 5
  • HTML 5- современый стандарт, поддерживающий мультимедия и семантические элименты

Основные Элементы HTML

HTMl состоит из различных тегов, которые задают структуру документа:

  1. Заголовки: теги < h1> ---< h6>.
  2. Параграфы: тег < p>
  3. Списки: маркированые < ul> и нумированые < ol>
  4. Гипперссылки: тег < a>

Пример кода

Простейшая структура HTML-документа:

!DOCTYPE html>
< html lang="ru">
< head>
< meta charset="utf-8">
< title>пример< /title>
< /head>
< body>
< h1>Привет, мир!< /h1>
< /body>
< /html>

эта страница создана для демонстарции базового HTML, без использования CSS
